  • Explain what a regex pattern is and why it’s useful
  • Use stringr to detect, extract, and replace patterns in text
  • Use regex to clean messy biological metadata (using palmerpenguins)
  • Handle missing values intentionally
  • Detect and remove exact duplicate rows
  • Select columns by type and by name patterns
  • Compute summaries across many columns without typing every column name
  • Create new classification columns using clear rules
  • Combine information from multiple tables using joins
